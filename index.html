Frequently Asked Questions

Does CodeshipAI access my code or repositories? +
No, CodeshipAI analyzes your code locally in the browser without sending it to our servers or requiring repository access. Your code stays private and secure.
Which browser-based IDEs are supported? +
CodeshipAI works with GitHub Codespaces, StackBlitz, CodePen, Replit, JSFiddle, and most other web-based development environments. New platforms are added regularly.
Can I customize the coding rules and standards? +
Yes! Pro and Team plans include custom rule configuration. You can set your own coding standards, disable specific checks, or add team-specific requirements.
How does the security vulnerability detection work? +
Our AI engine analyzes your code patterns against known vulnerability databases and security best practices. It identifies issues like XSS, SQL injection, insecure dependencies, and more.
Is there a free trial for paid plans? +
Yes! All paid plans include a 14-day free trial with full access to premium features. No credit card required to start your trial.
